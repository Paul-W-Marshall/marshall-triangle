## Overview

The Marshall Triangle positions three competing concerns at edge midpoints of an equilateral triangle:

- **Privacy (Red)** — User data protection and anonymity
- **Performance (Green)** — System speed and efficiency  
- **Personalization (Blue)** — Tailored user experiences

Secondary relationships emerge at vertices through additive color mixing, while perfect balance manifests as white light at the geometric center. The visualization uses Gaussian falloff (canonical sigma = 0.30) for smooth gradient transitions.
